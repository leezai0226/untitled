# 로그인 페이지 이용약관 및 개인정보처리방침 링크 연결 요청

로그인(또는 회원가입) 화면 하단에 있는 '이용약관'과 '개인정보처리방침' 텍스트에 클릭 가능한 링크를 연결하고, 해당 페이지들을 생성해 줘.

## 1. 로그인/회원가입 컴포넌트 수정
- 로그인 UI가 있는 파일(예: `/app/login/page.tsx` 또는 관련 Auth 컴포넌트)을 찾아서, 해당 텍스트를 Next.js의 `<Link>` 컴포넌트로 감싸 줘.
- '이용약관'은 `href="/terms"` 로, '개인정보처리방침'은 `href="/privacy"` 로 경로를 지정해 줘.
- 사용자가 클릭할 수 있는 링크임을 명확히 알 수 있도록, 마우스를 올렸을 때(hover) 밑줄이 생기거나 텍스트 색상이 살짝 진해지도록 Tailwind CSS를 적용해 줘.

## 2. 약관 및 방침 페이지(Placeholder) 신규 생성
- 라우팅 오류가 나지 않도록 `/app/terms/page.tsx` 와 `/app/privacy/page.tsx` 파일을 각각 새로 생성해 줘.
- 각 페이지 상단에는 "이용약관" / "개인정보처리방침" 이라는 큰 제목(Heading)을 넣어 줘.
- 본문에는 임시로 "여기에 내용이 들어갑니다."라는 텍스트를 넣고, 화면 왼쪽 위나 아래에 이전 페이지(또는 홈)로 돌아갈 수 있는 '[뒤로 가기]' 버튼을 깔끔하게 배치해 줘.
