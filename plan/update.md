# 클래스 가격 수정 및 연계 할인 로직 업데이트 요청

클래스의 정가와 초급반 수강생 대상 할인 금액을 아래와 같이 최신화해 줘.

## 1. 가격 체계 수정
- **초급반:** 99,000원 (유지)
- **중급반:** 129,000원 (기존 14.9만 원에서 하향 조정)

## 2. 연계 할인 로직 수정
- **할인 대상:** 초급반 수강 완료 이력이 있는 사용자
- **할인 금액:** 20,000원 할인 적용
- **최종 할인가:** 중급반 결제 시 **109,000원**으로 결제되도록 설정

## 3. 코드 반영 사항
- **프론트엔드:** 중급반 상세 페이지 및 결제 팝업에서 초급반 수강생에게는 109,000원이 노출되도록 수정해 줘.
- **백엔드 검증 (`/api/portone/verify/route.ts`):** - 서버에서 결제 위변조 검증 시, 초급반 수강생의 중급반 결제 기준 금액을 109,000원으로 업데이트해서 검증해 줘.
  - 일반 구매자의 중급반 결제 기준 금액은 129,000원으로 검증해 줘.
