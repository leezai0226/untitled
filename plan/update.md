# 초급반 수강생 대상 중급반 자동 할인 로직 구현 요청

초급반을 이미 구매한 수강생이 중급반을 추가로 결제할 때, 약속된 2만 원 할인이 자동으로 적용되도록 시스템을 구성해 줘.

## 1. 가격 및 할인 기준 정의
- **초급반 (Elementary):** 89,000원
- **중급반 (Intermediate):** 129,000원
- **올인원 패키지 (Package):** 198,000원
- **연계 할인:** 초급반 수강 이력이 있는 경우, 중급반 단품 결제 시 **20,000원 할인** (최종가 109,000원)

## 2. 수강 이력 확인 및 UI 제어
- 사용자가 중급반 페이지에 접속하면, Supabase의 `Orders` 테이블을 조회하여 해당 사용자가 '초급반'을 결제 완료(status: 'completed')했는지 확인해 줘.
- **초급반 수강생인 경우:** - 중급반 가격을 109,000원으로 자동 변경하여 표시해 줘.
  - "초급반 수강생 전용 2만원 할인 적용 중"이라는 안내 메시지를 노출해 줘.
  - 이미 초급반을 수강 중이므로, '올인원 패키지' 상품은 화면에서 숨기거나 비활성화 처리해 줘.

## 3. 결제 및 검증 로직 고도화
- **포트원 결제 요청:** 할인 대상자일 경우 `IMP.request_pay`에 전달하는 `amount` 값을 109,000원으로 설정해 줘.
- **서버 검증 (`/api/portone/verify/route.ts`):** - 서버단에서 결제 위변조를 검증할 때도 사용자의 초급반 수강 여부를 직접 다시 조회해 줘.
  - 수강 이력이 확인되면 실제 결제 금액이 109,000원인지 확인하고, 이력이 없다면 정가인 129,000원인지 확인하도록 검증 로직을 분기 처리해 줘.

## 4. 중급반 결제란 할인 안내
- 초급반과 같이 결제할 경우 199,000원에 진행 가능하다는 내용을 넣어줘.
