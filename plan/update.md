# Shop 상품 상세 페이지 4:5 미디어 슬라이더(사진+동영상) 기능 추가 요청

상품 상세 페이지 상단에 사진과 동영상을 동시에 지원하는 4:5 비율의 슬라이더 기능을 추가하고, 관리자 페이지에서 이를 관리할 수 있도록 구현해 줘.

## 1. 데이터베이스 (Supabase) 및 저장소 설정
- `products` 테이블에 슬라이더 미디어 정보를 저장할 수 있는 `slider_media` 컬럼(JSONB 타입)을 추가해 줘. 
- 각 항목은 `{ url: string, type: 'image' | 'video' }` 형태의 객체 배열로 저장되어야 해.
- Supabase Storage에 영상 파일도 업로드할 수 있도록 허용 용량 및 파일 형식을 체크해 줘.

## 2. 관리자 페이지 UI 수정 (`/app/admin/products/...`)
- 상품 등록 및 수정 폼에서 **'상세 이미지' 섹션 바로 위**에 [슬라이더 미디어 추가] 영역을 만들어 줘.
- 이미지(.jpg, .png, .webp)와 동영상(.mp4, .mov) 파일을 모두 업로드할 수 있어야 해.
- 드래그 앤 드롭으로 순서를 변경할 수 있거나, 업로드된 순서대로 슬라이더에 노출되게 해 줘.
- 상세 이미지와 마찬가지로, **파일을 추가하지 않으면 화면에 슬라이더 영역 자체가 나타나지 않도록** 예외 처리를 해 줘.

## 3. 프론트엔드 Shop 상세 페이지 UI (`/app/shop/[id]/page.tsx`)
- 상품 상세 페이지 상단에 미디어 슬라이더를 배치해 줘.
- **비율 설정:** 모든 미디어(사진, 영상)는 **4:5 비율 (`aspect-[4/5]`)**로 고정되어야 하며, 영역에 꽉 차도록 `object-cover` 설정을 적용해 줘.
- **동영상 처리:** 미디어 타입이 'video'인 경우, 자동으로 재생(`autoplay`), 반복(`loop`), 음소거(`muted`), 인라인 재생(`playsinline`) 속성을 가진 `<video>` 태그로 렌더링해 줘.
- **슬라이드 라이브러리:** `Swiper.js` 또는 `Framer Motion`을 사용하여 부드러운 스와이프 효과와 좌우 이동 화살표, 페이지네이션(점 표시)을 넣어 줘.

## 4. 조건부 렌더링
- `slider_media` 데이터가 비어있거나 null인 상품의 경우, 슬라이더 영역을 완전히 숨겨서 레이아웃이 깨지지 않게 해 줘.
