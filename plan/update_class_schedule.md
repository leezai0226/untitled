# 클래스 페이지 강의 일정 업데이트 요청

클래스 예약 페이지(`/app/class/page.tsx` 또는 관련 데이터 설정 파일)에 노출되는 강의 일정을 아래와 같이 최신 정보로 수정해 줘.

## 1. 브이로그반 (Vlog Class) 일정 변경
- 기존 일정: 4월 7일 14-18시, 4월 11일 15-19시
- **새로운 일정:**
  - 4월 21일 14:00 - 18:00
  - 4월 25일 14:00 - 18:00

## 2. 숏폼반 (Short-form Class) 일정 변경
- 기존 일정: 4월 21일 14-18시, 4월 25일 14-18시
- **새로운 일정:**
  - 4월 26일 14:00 - 18:00
  - 4월 28일 14:00 - 18:00

## 3. 수정 가이드
- 화면에 보이는 날짜 텍스트뿐만 아니라, 수강생이 일정을 선택할 때 백엔드(Supabase 등)로 전달되는 데이터 값도 해당 날짜에 맞춰서 함께 업데이트해 줘.
- 변경된 일정이 주말인지 평일인지 확인하여 UI상에 요일 정보가 포함되어 있다면 정확한 요일로 기재해 줘.
