# 관리자 페이지 상품별 자주 묻는 질문(FAQ) 관리 기능 추가 요청

shop 페이지에 상품(디지털 에셋, 클래스 등)을 등록하거나 수정할 때, 관리자가 해당 상품에 대한 '자주 묻는 질문(FAQ)' 목록을 동적으로 추가하고 관리할 수 있는 기능을 구현해 줘. 

## 1. 데이터베이스 (Supabase) 구조 업데이트
- 기존 상품(Products) 테이블에 FAQ 데이터를 저장할 수 있도록 컬럼을 추가해줘.(예: JSONB 타입의 `faqs` 컬럼, 형태는 Class 페이지 FAQ랑 동일하게) 
- FAQ 데이터 구조는 질문(Question)과 답변(Answer)의 쌍으로 이루어져야 해.

## 2. 관리자 페이지 UI 수정 (`/app/admin/products/...`)
- 상품 등록 및 수정 페이지의 폼(Form) 영역 하단에 **[자주 묻는 질문 추가]** 버튼을 만들어 줘.
- 이 버튼을 누르면 '질문(Q)'과 '답변(A)'을 입력할 수 있는 입력 칸이 동적으로 한 세트씩 추가되게 해 줘.
- 추가된 FAQ 세트를 삭제할 수 있는 [삭제] 버튼도 각 세트 옆에 배치해 줘.
- 폼을 제출(Submit)할 때, 입력된 FAQ 배열 데이터가 상품 정보와 함께 Supabase에 안전하게 저장 및 업데이트 되도록 로직을 작성해 줘.

## 3. 프론트엔드 Shop 상세 페이지 UI 반영 (`/app/shop/[id]/page.tsx` 등)
- 고객이 상품 상세 페이지를 볼 때, 해당 상품에 등록된 FAQ 데이터가 있다면 화면 하단(또는 상세 설명 아래)에 노출해 줘.
- 디자인은 제목(Q)을 클릭하면 아래로 내용(A)이 펼쳐지는 **아코디언(Accordion)** 형태의 UI로 깔끔하게 구현해 줘. (Tailwind CSS 활용)
- 만약 등록된 FAQ가 없는 상품이라면, 이 영역이 자연스럽게 숨겨지도록(렌더링되지 않도록) 예외 처리를 해 줘.
